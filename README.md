  Bolsa de Valores (em andamento)

  Projeto pessoal desenvolvido com Java Spring Boot no backend e React no frontend para acompanhamento de ações da bolsa de valores. O sistema permite ao usuário criar alertas personalizados de preço e receber notificações em tempo real pela aplicação e pelo Discord. Utiliza PostgreSQL para persistência de dados e Kafka para processamento assíncrono de eventos.
